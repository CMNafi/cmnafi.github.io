Manager Search

A manager intelligence dashboard that ties together entity resolution, Form ADV data, 13F holdings, and narrative summaries.

Started March 18, 2026 Updated March 18, 2026
Next.jsTypeScriptSEC APIOpenAIForm ADVForm 13F

Problem

Public filings contain the right answers, but the path from manager name to a usable diligence snapshot is still too fragmented and manual.

Solution

Build one dashboard that resolves entities, gathers key regulatory and holdings data, prunes the raw inputs responsibly, and produces a readable summary for faster research.

Who it's for

Investors, allocators, analysts, and diligence teams who need a faster path from name lookup to a trustworthy manager snapshot.

Manager Search is a product direction that sits directly between public regulatory data and a real research workflow.

Why it matters

The work is less about making filings look pretty and more about making them usable. A name should resolve into a manager profile quickly, the key identifiers should be obvious, and the holdings view should support the next decision instead of creating more cleanup work.

Key features

  • Debounced entity search with CRD and CIK resolution.
  • Deep links to SEC EDGAR and IAPD profiles.
  • Top-holdings rollup from 13F data with deterministic value calculations.
  • LLM summary generation using only pruned and labeled inputs.

Roadmap

  • Add comparison mode for multiple managers.
  • Surface filing change detection between quarters.
  • Expand into a broader manager research platform.
