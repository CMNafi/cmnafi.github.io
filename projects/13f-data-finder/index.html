13F Data Finder

A searchable entry point into holdings data, manager behavior, and repeatable research questions.

Started January 22, 2026 Updated March 12, 2026
AstroTypeScriptStatic contentData pipelines

Problem

Holdings data is abundant, but pattern-finding across managers and quarters is still too manual.

Solution

Create a public overview and eventually a separate app experience that makes manager discovery, comparison, and monitoring much more fluid.

Who it's for

Investors, researchers, and diligence teams who want to move from raw filings to structured insight more quickly.

This project is driven by a simple product question: what would 13F research feel like if it were designed as a first-class workflow instead of a patchwork?

Direction

The first phase is content-first. The public site explains the intent, design logic, and roadmap before the eventual app becomes public.

Key features

  • Discover managers by strategy and holdings behavior.
  • Monitor quarterly changes without stitching together multiple tools.
  • Create a clearer foundation for deeper analytics later.

Roadmap

  • Define the public taxonomy and project narrative.
  • Add screenshots or visual placeholders for the app surface.
  • Expand toward change tracking and manager exploration modules.
