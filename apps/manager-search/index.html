Manager intelligence

Manager Search

A dedicated dashboard for institutional manager resolution, public filing analysis, and AI-assisted narrative summaries.

This app is designed for public-market research workflows that start with a name and need to end with a cleaner picture of the manager, the filings, and the portfolio.

Back to apps Browse projects

Core capabilities

  • Resolve firms through SEC CIK and FINRA CRD identifiers.
  • Pull the latest Form ADV operating profile and most recent 13F cover page data.
  • Collapse raw holdings into a ranked top-positions view instead of sending unfiltered JSON to an LLM.
  • Generate a concise narrative summary for diligence, research, and manager triage.

Workflow

  1. Search for a manager name or identifier.
  2. Confirm the matched entity and open the SEC / IAPD links.
  3. Review the top holdings, operational profile, and filing status.
  4. Read an AI-generated narrative built only from pruned and labeled inputs.
